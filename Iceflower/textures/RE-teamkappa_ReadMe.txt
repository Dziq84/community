For RE style do:
1. Normal RE style procedure
2. Center the shadow level(horizontal and vertical)
3. Select the diffuse level and there grey gradient part.
4. Go to "Colours"/ "Farben" -> "Brightness-Contrast"/ "Helligkeit/ Kontrast" -> "Edit these settings as levels"/ "Diese Einstellungen als Werte bearbeiten"
5. Change Channels, Input Levels, Output Levels like the chart as below and use it

			 (Input Levels)   (Output Levels)
   Channel -  Black - White -  Black - White   
	Red	   -  140   -  240  -  145   -  199
	Green  -  140   -  240  -  146	 -  200
	Blue   -  140   -  240  -  147   -  202
